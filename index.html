Info:

This website has been developed as an independent continuation of my previous project created for the Aspiring Scientists Summer Internship Program (ASSIP) at George Mason University, and I hope that it will be useful to anyone interested in protein modeling and 3D printing. This web app uses a custom Python script in order to convert .pdb files, commonly sourced from The Research Collaboratory for Structural Bioinformatics (RCSB) Protein Data Bank (PDB), to .scad files. These can be used in the OpenSCAD application to create files for 3D printing or other 3D modeling applications.
